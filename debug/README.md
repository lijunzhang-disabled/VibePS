# PS1 Debug Notes

This directory mirrors the `../nds/debug` workflow. Add investigation notes here
when a compatibility issue needs more than a small code comment.

Use this shape for new notes:

```text
# YYYY-MM-DD_short-title

## Symptom

## Hypothesis

## Evidence

## Fix

## Regression
```

Early expected notes:

- BIOS boot trace and first exception mismatch
- R3000A load-delay and branch-delay edge cases
- GPU DMA linked-list ordering table behavior
- CD-ROM response/data queue timing
