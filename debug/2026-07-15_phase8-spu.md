# Phase 8 SPU Baseline

Implemented the PS1 SPU as a clocked device rather than a register scaffold.

## Coverage

- 512 KiB sound RAM, manual transfer FIFO, DMA4 read/write, status, and IRQ9
- 24 SPU-ADPCM voices with pitch, PMON, loop flags, ADSR, volume, and noise
- 44.1 kHz stereo mixer, CD input/capture, voice capture, and reverb processing
- Core drain API and frontend streaming WAV output
- Native tests for decode ordering, pitch modulation, ADSR, looping, transfers,
  sample cadence, voice/CD mixing, capture, DMA4, MMIO, and interrupt routing

## Deferred Accuracy

- Hardware Gaussian interpolation instead of the current linear interpolation
- Exact 44.1 kHz register application and sound-RAM arbitration order
- Inactive voice reads and less common transfer/reverb bus edge cases
- XA/CD-DA decoding in the CD-ROM device and realtime host audio output

Primary reference: https://psx-spx.consoledev.net/soundprocessingunitspu/
